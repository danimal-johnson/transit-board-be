Quickstart

API Key

None required. This is a read-only API. As long as it isn't abused, it can remain open to the public.

Tips

  • All requests are of type GET
  • All endpoints start with the prefix of /api
  • To see the full documentation, including motivations and use cases, view the project on GitHub.

Some examples to get you going

Agency info, calendar range covered, current date and time:
/api/info
All the transit routes served by this API:
/api/routes
Details about a specific route:
/api/routes/91
All the stops on a route:
/api/stops?route=01
Details about a specific stop:
/api/stops/02507
All of today's departures for a stop:
/api/departures?stop=02507&date=today
Departures for a stop, date, and route:
/api/departures?stop=02507&date=20230624&route=01
